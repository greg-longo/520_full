    💻 The Full Clean Pipeline - What an Expert Would Write
    eagles_clean.py - complete cleaning script
    import pandas as pd
import numpy as np

df = pd.read_csv('eagles_players.csv')

# 1. Standardize column names first (makes everything below easier)
df.columns = df.columns.str.lower().str.replace(' ', '_')

# 2. Drop exact duplicate rows
df = df.drop_duplicates()

# 3. Convert salary: strip "$" and "," then cast to float
df['salary'] = (
    df['salary']
    .str.replace('$', '', regex=False)
    .str.replace(',', '', regex=False)
    .astype(float)
)

# 4. Fix invalid home_runs value (-3 is impossible)
df.loc[df['home_runs'] < 0, 'home_runs'] = np.nan

# 5. Impute missing numeric values with median
for col in ['age', 'home_runs', 'games_played']:
    df[col].fillna(df[col].median(), inplace=True)

# 6. For batting_avg: impute + add missingness flag
df['batting_avg_missing'] = df['batting_avg'].isna().astype(int)
df['batting_avg'].fillna(df['batting_avg'].median(), inplace=True)

# 7. Keep Kim's salary as-is (confirmed real data)
# Optional: log-scale salary for model input
df['log_salary'] = np.log1p(df['salary'])

print(f"Clean dataset: {df.shape[0]} rows, {df.shape[1]} columns")
df.to_csv('eagles_clean.csv', index=False)

    🧠 What This Simulation Covers

    1. Missing data strategy matters. Dropping rows loses information. Filling with zero introduces false signals. Median imputation is robust. Adding a missingness flag is an advanced technique that lets the model learn from the fact that the value was unknown.
    2. Duplicates silently bias models. If 11 rows are duplicated, those players are effectively oversampled - the model sees them more often and learns to weight their pattern more heavily. Always check for duplicates before training.
    3. Data types are not automatic. pandas reads what it sees. A "$" in a numeric column forces it to read the whole column as strings. Type conversion is a required, not optional, step.
    4. Not all outliers are errors. Kim's salary is real. Blindly clipping or dropping outliers removes valid signal. Always try to confirm with the source before modifying extreme values.
    5. Column naming conventions prevent bugs. Spaces in column names prevent dot notation and cause subtle errors in pipelines. Standardizing to lowercase underscore style is a best practice you should apply to every dataset before doing anything else.
