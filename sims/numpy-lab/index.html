An interactive NumPy lab

You have been handed access to StreamChart's internal dataset - 10 songs from five artists whose names you may recognize. Your job is to work through six coding challenges covering the core of what makes NumPy worth learning: how it stores data, how to slice and index into arrays, how to run math across an entire dataset without writing a loop, and how to filter rows with conditions.

The dataset stays visible on the left the whole time. On the right you will see the challenge description, a code editor with starter code already filled in, and an output panel where your results appear. Each challenge asks you to complete or modify a few lines to produce a specific output.

NumPy Reference Card
Array Creation
np.array([1, 2, 3])
Create array from a Python list
np.zeros(5)  /  np.ones(5)
Array of 0s or 1s, length 5
np.arange(0, 10, 2)
Range [0, 2, 4, 6, 8] with step
np.linspace(0, 1, 5)
5 evenly spaced values from 0 to 1
Array Attributes
arr.shape
Dimensions tuple — (10,) or (5,2)
arr.dtype
Data type: int64, float64, <U30, etc.
arr.ndim  /  arr.size
Number of dimensions / total elements
Indexing & Slicing (1D)
arr[0]  /  arr[-1]
First / last element
arr[1:4]
Elements at index 1, 2, 3 (stop is exclusive)
arr[:3]  /  arr[3:]
First 3 elements / index 3 to end
arr[::2]
Every other element (step of 2)
Indexing (2D)
arr[2, 1]
Row 2, column 1
arr[2]  or  arr[2, :]
Entire row 2
arr[:, 0]
Column 0 across all rows
Boolean Indexing
arr[arr > 5]
Elements greater than 5
arr[(arr > 2) & (arr < 8)]
Multiple conditions — use & not and
arr[(arr < 1) | (arr > 9)]
Either condition — use | not or
Fancy Indexing
arr[[0, 3, 1]]
Elements at indices 0, 3, 1 in that order
idx = [2, 5, 0]; arr[idx]
Use a variable as the index list
Vectorized Math
arr * 2  /  arr + 10  /  arr ** 2
Applied to every element — no loop needed
arr1 + arr2
Element-wise addition of two arrays
Statistical Methods
np.mean(arr)  /  np.median(arr)
Mean and median
np.std(arr)  /  np.var(arr)
Standard deviation and variance
np.max(arr)  /  np.min(arr)
Maximum and minimum value
np.sum(arr)
Sum of all elements
np.percentile(arr, 75)
75th percentile
np.sort(arr)  /  np.argsort(arr)
Sorted values / indices that would sort it
Display Tip
np.set_printoptions(linewidth=200)
Prevent arrays from wrapping to multiple lines
