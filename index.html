About This Course

DTSC 520 is an introductory graduate course in data science at Eastern University. No prior programming experience is assumed. By the end of the semester you'll be able to wrangle real datasets, write Python programs, use version control professionally, and communicate findings to a non-technical audience.

What You'll Build

  • A personal GitHub portfolio with at least 3 data projects
  • Data cleaning and analysis scripts in Python and pandas
  • Visualizations that tell a clear story from messy data
